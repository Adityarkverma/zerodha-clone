# Zerodha Clone (MERN Stack)

A full stack trading platform clone inspired by Zerodha built using the MERN stack.

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## Features
- User Signup & Login Authentication
- Dashboard with Holdings, Positions, Orders
- Buy & Sell Stocks
- Responsive UI
- Secure JWT Authentication

## Project Structure
backend → API and database  
frontend → Landing website  
dashboard → Trading dashboard

## Installation

Clone the repository

git clone https://github.com/Adityarkverma/zerodha-clone.git

Install dependencies

cd backend
npm install

cd frontend
npm install

cd dashboard
npm install

Start servers

npm start
